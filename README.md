# raspberrypi - Growe Water Level Sensor
### Bash script to convert raw i2c data from Growe water level sensor into % value
https://wiki.seeedstudio.com/Grove-Water-Level-Sensor/

### Pins connection
    VCC - 3V3 PIN (or 5V)
    GND - Ground PIN
    SDA - I2C 1 SDATA
    SCL - I2C 1 SCLOCK



### run script
    ./liquidlevel.sh
